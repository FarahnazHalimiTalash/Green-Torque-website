# Green Torque Website

The official website for **Green Torque Auto Repair**, designed to provide customers with a modern, intuitive, and responsive online experience.

The platform serves as the company's digital presence, allowing customers to explore available automotive services, schedule appointments, access business information, and connect with the Green Torque team.

---

## Purpose

The Green Torque website is designed to:

* Showcase the company's automotive repair services.
* Simplify the appointment booking process.
* Improve customer accessibility.
* Strengthen Green Torque's online presence.
* Deliver a seamless experience across desktop, tablet, and mobile devices.

---

## Key Features

* Responsive single-page website
* Modern minimalistic interface
* Service catalog
* Appointment booking
* Accepted payment methods
* Business hours
* Contact information
* Interactive location map
* Mobile-first navigation

## Getting Started

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

---

## Customization

Update the following placeholders in `index.html` with your real business details:

* Address, phone, and email in the **Contact** section
* Google Maps embed URL in the map iframe
* Business hours in the **Booking** section
* Stats and about copy as needed

Replace `images/logo.svg` with your official Green Torque logo when available.
